raspbirra

Frontend library for developing the visual interface of the raspbirra system, using canvas library RaphaelJS.

Parameters
div_id (string) Html id from the element that will have the canvas
width (number) Canvas width
height (number) Canvas height
Returns
raspbirra: Frontend object for drawing and handle the raspbirra human machine interface (HMI)

rb_element

Generic raspbirra element Object (rb_element) for interactuate with kettles, coils, valves,...

rb_element(_id: string, _name: string, _stat: number, _coords: rb_coords, _coords_in: rb_coords, _coords_out: rb_coords, _bidirectional: any, _ch_color_function: function, _ch_text_function: function, bidirectional: bool)
Parameters
_id (string) rb_element ID
_name (string) rb_element name
_stat (number) Stat of the element (0- Closed / 1- Openned)
_coords (rb_coords) Coords of the rb_element
_coords_in (rb_coords) Coords of the thread in of the rb_element
_coords_out (rb_coords) Coords of the thread out of the rb_element
_bidirectional (any)
_ch_color_function (function) Callback function that will be called for change the color of the rb_element
_ch_text_function (function) Callback function that will be called for change the text of the rb_element
bidirectional (bool) If True, will permit liquid flow for two directions of the element
